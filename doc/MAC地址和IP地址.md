## MAC地址和IP地址

计算机有IP地址，网卡有MAC地址。

IP地址全球同一规划，并且每个IP地址都是全球唯一的，

IP地址分网络部分和主机部分：

子网掩码就是用来告诉我们哪里是主机部分，哪里是网络部分。 **有多少个255，就代表哪部分是网络部分**。
例如：下面的13是网络部分，0.0.0是主机部分

```
IP地址：13.0.0.0
子网掩码：255.0.0.0
```

MAC地址 - 物理地址：出厂的时候，就已经固定在网卡里，48位的二进制；每个接口都有MAC地址  

常看本机网卡的MAC地址的命令：`ipconfig /all`
